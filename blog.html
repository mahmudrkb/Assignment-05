1. What is the Document Object Model (DOM)?

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM that allows you to handle events efficiently by taking advantage of event bubbling. Instead of attaching event listeners to multiple child elements individually, you attach a single event listener to a parent element. The event listener on the parent can "delegate" event handling to its child elements, based on the specific target of the event.

Instead of attaching many event listeners to individual child elements, you attach just one to the parent. This reduces memory usage and speeds up event handling, especially in cases with a large number of child elements.

4. How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles in the DOM using JavaScript. The DOM provides methods and properties that allow you to read, update, add, or remove attributes and styles dynamically.
